# 📱 VibeCoding项目说明文档

## 🏆 赛题选择

**知识奇点** & **智享生活**

## 📝 项目名称

**智能学习助手APP**

## 🎯 项目概述

一个学习助手APP，支持Android，功能包括**待办备忘、计时器、番茄钟、思维导图、笔记、事件倒数日、智能AI助手**和**电子木鱼**等功能。

## 💡 项目创意与优势

### 🎨 创意来源

在当今快节奏的学习环境中，学生们需要同时管理多项任务：完成作业、准备考试、参与项目等。我们的团队注意到，虽然市面上存在各种单一功能的学习工具，但缺乏一个真正集成且智能化的学习助手。

我们以"知识奇点"和"智享生活"为核心理念，旨在打造一个全能型学习助手。应用名称"VibeCoding"寓意着在轻松愉快的氛围中高效学习和编程。

### 🌟 核心优势

#### 🔧 多功能一体化
与市面上单一功能的应用不同，我们将8个核心学习功能集成在一个应用中：
- **待办事项管理**：帮助您规划每日学习任务，避免遗漏重要作业
- **计时器与番茄钟**：结合时间管理理论，提高学习专注度
- **思维导图**：可视化知识结构，助力理解和记忆
- **笔记系统**：随时随地记录灵感和知识点
- **倒数日提醒**：重要考试或事件的贴心提醒
- **AI智能助手**：基于DeepSeek的强大AI，快速解答学习问题
- **电子木鱼**：独特的放松功能，缓解学习压力

#### 🤖 强大的AI集成
我们的AI助手采用DeepSeek API，具有以下优势：
- 响应速度快，避免"服务器繁忙"的尴尬
- 支持复制AI回答内容，便于整理学习资料
- 专注学习领域，提供专业的学习建议和问题解答

#### 🎯 用户体验优化
- 简洁直观的界面设计，降低学习成本
- 丰富的交互反馈和引导提示
- 每个功能模块都有贴心的使用建议（以">"开头的提示）

#### 📱 贴心的功能设计
- **本地存储**：保障数据安全，无需担心网络问题
- **跨平台支持**：基于Flutter开发，未来可轻松扩展至iOS等平台
- **个性化设置**：各项计时功能均可自定义时间设置

#### 🎁 独特的附加功能
- **电子木鱼**：在紧张的学习之余，提供轻松有趣的减压方式，同时也是一种独特的专注力训练工具

### 🎯 目标用户群体
- 大中学生：需要管理大量学习任务和时间的群体
- 自学者：希望通过科学方法提高学习效率的人群
- 教育工作者：需要整理知识结构和教学内容的老师

### 🚀 技术优势
- 使用Flutter框架，确保应用性能和跨平台兼容性
- 采用本地存储方案，保障用户数据隐私和应用响应速度
- 模块化设计，便于功能扩展和维护

## 🌟 功能介绍

### 📋 4-1 待办事项

点击首页的`待办事项`模块进入。

点击右下角的`+`可以创建新的待办事项。

点击现有待办事项左边的圆圈可以将待办事项标记为完成。再次点击可以取消标记。

点击右侧的`删除`可以删除待办事项。

> 可有效防止到ddl忘交作业痛失平时分的情况

### ⏱️ 4-2 计时器

点击首页的`计时器`模块进入。

点击下方的`设置时间`可设置要计多长时间。

点击`开始`开始计时，点击`重置`恢复到当前设定的时间。

> 最后刷五分钟视频就学习

### 🍅 4-3 番茄钟

点击首页的`专注计时`模块进入。

基于著名的番茄工作法的时间管理工具，默认专注-学习周期为25分钟-5分钟，可通过点击`设置专注时间`和`设置休息时间`来调整。

`重置`会重置到一个专注-休息周期的开头。

> 每四个"番茄"后，另外休息15-30分。

### 🧠 4-4 思维导图

点击首页的`思维导图`模块进入。

点击一个主题块选中。

点击右下角的`+`在当前选中主题块下添加子主题块。

双击或点击左下角的`编辑`对当前选中主题块内容进行编辑。

点击下方的`删除`删除当前选中主题块。

点击右下角的`保存`保存当前思维导图。

点击上方的第一个按钮`撤销`撤销上一步操作。

点击上方的第二个按钮`查看记录`查看已保存的思维导图，可以导入。

点击上方的第三个按钮`+`新建思维导图（会覆盖当前思维导图，请确认已经保存）。

点击上方的第四个按钮`分享`可以分享思维导图。

点击上方的第五个按钮`整理`可以一键整理为树状思维导图。

> 以后政治再有思维导图可以用这个写了

### 📝 4-5 笔记

点击首页的`笔记本`模块进入。

点击右下角的`+`添加新笔记。

> 课堂记录、整理结论的好地方

> 上学期我高数整理结论还是直接打在微信聊天框里的，有这个就不用了

### 📅 4-6 倒数日

点击首页的`倒数日`模块进入。

点击右上角的`+`按钮添加新的倒数日事件。

点击现有的倒数日事件右侧的`三点`，可以同步到手机日历设置日程，也可以删除。

> 抗战胜利80周年阅兵

调用满血版DeepSeek，并且输出很快，不会`服务器繁忙，请稍后再试`。

支持输出内容复制。

## ⚙️ 运行指南

### 💻 方法一：使用预编译APK

- 我们已在仓库中提供了预编译的APK文件，可直接安装使用。
- 可以在电脑上使用安卓模拟器运行，或复制到Android手机上安装运行。

### 🔧 方法二：从源代码构建

1. 确保已安装Flutter开发环境：
   - 下载并安装Flutter SDK
   - 配置环境变量
   - 安装Android Studio或VS Code Flutter插件

2. 克隆或下载项目源代码：
   
   
   
3. 进入项目目录并获取依赖：
   ```
   cd mobile_app
   flutter pub get
   ```

4. 连接Android设备或启动模拟器

5. 构建APK：
   ```
   flutter build apk --debug
   ```
   或构建发布版本：
   ```
   flutter build apk --release
   ```

6. 构建完成后，APK文件位于：
   ```
   build\app\outputs\flutter-apk\app-debug.apk
   ```
   或
   ```
   build\app\outputs\flutter-apk\app-release.apk
   ```

7. 将APK文件复制到Android设备上安装运行
